LogManagingSystem

Secure and Efficient Log Management with Kotlin and Go

Overview

LogManagingSystem is a robust and secure log management solution designed for applications requiring reliable log collection and storage. It leverages Kotlin for client-side operations and Go for the backend, ensuring both efficiency and security.

Key Features:

  • Encrypted Log Transmission: Utilizes AES encryption to secure log data during transmission.
  • Secure Authentication: Implements basic authentication to protect API endpoints.
  • Organized Log Storage: Stores logs in directories based on device identifiers (IMEL), making retrieval efficient.
  • High-Performance Backend: Go backend ensures fast and efficient log processing and storage.
  • Cross-Platform Compatibility: Kotlin client enables easy integration with various platforms.

Kotlin Client Usage

To use the Kotlin client, follow these steps:

  1. Add Dependency: Include the LogManagingKotlinLib in your Kotlin project.
  2. Initialize the Library: Instantiate LogManagingKotlinLib with your API endpoint, username, password, and encryption key.
  3. Send Encrypted Logs: Use the sendEncryptedLog function to send log messages.

Example:


import kotlinx.coroutines.runBlocking

fun main() = runBlocking {
    val logManagingKotlinLib = LogManagingKotlinLib(
        "http://<api_host>:<api_port>",
        "myuser",
        "mypassword",
        "your32bytekeyhere!"
    )

    val result = logManagingKotlinLib.sendEncryptedLog("Your Log Message", "LogTag")

    result.fold(
        onSuccess = { message ->
            println("Log sent successfully: $message")
        },
        onFailure = { exception ->
            println("Error sending log: ${exception.message}")
            exception.printStackTrace()
        }
    )
}

Note: Replace <api_host>, <api_port>, myuser, mypassword, and your32bytekeyhere! with your actual values.

Go Backend Setup and Usage

The Go backend provides a secure API for receiving encrypted logs. To set up and use the Go backend:

  1. Clone the Repository: Clone the LogManagingSystem repository from GitHub.
  2. Configure Environment: Set the username, password, and encryptionKey variables in the main.go file.
  3. Run the Server: Execute go run main.go to start the server.
  4. Send Logs: Use a client (like the provided Kotlin client or curl) to send encrypted logs to the server's endpoint.

Example using curl:


curl -X POST -H "Authorization: Basic <encoded_credentials>" -H "Content-Type: application/json" -d '{"encrypted_data":"<encrypted_data>"}' http://localhost:80/

Note: Replace <encoded_credentials> and <encrypted_data> with your actual encoded credentials and encrypted log data.
